1) IndentationError - ошибка, которая возникает в случае неправильного отступа. Для воспроизведения необходимо пропустить отступ.
Пример:
number = int(input("Введите число - "))
if number > 10:
print("yes")
else:
    print("no")

Был пропущен отступ вначале строки "print("yes")".

2) NameError - ошибка возникает в случае, если пользователь указывает переменную, которая не найдена в локальной или глобальной области. Для ее воспроизведения необходимо указать имя несуществующей переменной.
Например:
number = 4
print(name)

Было указано имя несуществующей переменной "name".

3) SyntaxError - ошибка возникает в случае, если интерпретатор обнаружит синтаксическую проблему в коде. Для ее воспроизведения необходимо, например, не поставить двоеточие в конструкции условия:
Пример:
age = int(input("Введите возраст - "))
if 50 > age > 18
	print("yes")
else:
    print("no")

В данном примере был упущен знак ":" в строке "if 50 > age > 18".

4) ZeroDivisionError - ошибка возникает при делении на ноль. Для ее воспроизведения необходимо одним из значений вписать ноль
Пример:
number_1 = 20
number_2 = 0
result = number_1 / number_2
print(result)

В данном случае у нас делитель равен нулю.

5) ModuleNotFoundError - ошибка, которая возникает при попытке импортирования несуществующего модуля или не верно указан путь откуда модуль должен быть импортирован. Также она возникает при допущении ошибки в имени модуля.
Пример:
import week

В данном случае мы указали модуль с несуществующим именем "week".
